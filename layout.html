Vertical alignment

While not exclusive to responsive design, vertical alignment is certainly worth a mention. Since the birth of CSS, vertical alignment has been a thorn in the side of the UI developer because there was never a simple solution, and vertical-alignment just never seemed to live up to its name, at least not without headaches.

Flexbox finally introduces vertical alignment, and it does exactly what it says on the box!

I'm vertically aligned!

Border box

Arguably the best yet most subtle feature introduced by CSS3 was the border-box property. It was such a godsend for front-end developers that CSS Tricks decided to observe International Box-Sizing Awareness Day in February.

This adored property makes styling responsive layouts many times easier to accomplish by simply specfying that an elements padding and border should be calculated as a part of the total width and height.

My border, padding, width and height are calculated as one so I don't overflow and cause nasty scrollbars!

Growing blocks

Flexbox also allows us to grow elements using the flex-grow property. In this scenario we have three blocks, each of them are set to "flex-grow: 1". This causes all three blocks to fill the width of the container, taking an equal share of available space.
